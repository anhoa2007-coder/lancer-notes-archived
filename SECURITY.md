# Security Policy

## Supported Versions

Since there's many users who want to stay supported against latest security issues, here's the real list of the supported versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.4.2 - 1.3.8   | Yes (Full support) |
| 1.3.2 - 1.3.7   | Ending in July 8th |
| 1.0.0 - 1.2.1   | Not supported* |

*You shouldn't use anything below 1.2.1, because the XSS protection is not enforced fully and will get you to higher security risk. More importantly, the code is in much worse position that it will be hard to fix.
1.3.8 was kept supported due to being used as a website in Neocities.
## Reporting a Vulnerability

If you want to report any security vulnerabilities, check out [Issues](https://github.com/anhoa2007-coder/lancer-notes/issues) tab in the repo, then apply **security** label.
